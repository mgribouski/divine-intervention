# Divine Intervention Larp
This is the website for Divine Intervention, a larp running Spring-Fall 2017.
- - -
## Run Locally
```shell
npm install
bower install
npm run less-compile
```
Then run a webserver from this directory. For example, `python -m SimpleHTTPServer`.
